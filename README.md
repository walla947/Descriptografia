Programa em C# que descriptografa mensagens binárias interceptadas de uma rede de navio.

Exemplo de entrada: 10010110 11110111 01010110 00000001 00010111 00100110 01010111 00000001 00010111 01110110 01010111 00110110 11110111 11010111 01010111 00000011


Também foi feito uma versão em C++ em: [github.com/walla947?tab=repositories](https://github.com/walla947/DescriptografiaC-)
